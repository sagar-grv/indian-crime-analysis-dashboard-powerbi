# Dataset Setup

The Power BI template expects a CSV source with this structure:

```text
State,District,Year,Murder,Rape,Robbery,Theft,Kidnapping_Abduction,Total_IPC_Crimes
```

The original dataset is not included in this repository.

Before publishing or redistributing data:

1. identify the official dataset owner,
2. record the source URL and retrieval date,
3. confirm the reporting definitions and coverage,
4. add the applicable licence, and
5. verify that redistribution is permitted.

Do not use the template as evidence of the dataset's accuracy, completeness or current status.
